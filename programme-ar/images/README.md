# Images – Landing « Vendre un programme, pas un produit »

Placez dans ce dossier les images suivantes pour que la page s’affiche correctement.

## Fichiers attendus

| Fichier | Usage | Recommandations |
|--------|--------|------------------|
| `logo-vitamax.png` | Logo VITAMAX (header + footer) | PNG transparent, hauteur ~56–88px |
| `hero-marocaines.jpg` | Image hero (plein écran) | Femme(s) marocaine(s), peau/beauté, 1200×800 px ou plus |
| `temoignage-1.jpg` | Témoignage / avant-après 1 | Format paysage 4:3 |
| `temoignage-2.jpg` | Témoignage / avant-après 2 | Format paysage 4:3 |
| `temoignage-3.jpg` | Témoignage / avant-après 3 | Format paysage 4:3 |

## Placeholders

Si les fichiers sont absents, la page reste utilisable :
- Le logo peut être remplacé par le texte « VITAMAX » (géré en HTML).
- Le hero affiche un fond crème (CSS).
- Les blocs témoignages gardent une zone image avec fond neutre.

## Optimisation

- Nommez les fichiers sans espaces (utilisez des tirets).
- Compressez les JPG/PNG pour le web (outils type TinyPNG, ImageOptim).
- Préférez le format WebP si votre hébergement le sert correctement (en gardant JPG/PNG en fallback).
